# spaced-repetition

- Frontend: ReactJS, Redux, React-Bootstrap
- Backend: Node.js
- Database: PostgreSQL

## Setup

Requirement:

- postgreSQL 8.7.3
- node.js 14.17.2


In the project's client directory, run:

npm run start

In the project's server directory, run:

npm run start
